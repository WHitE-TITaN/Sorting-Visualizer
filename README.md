<h1> Sorting-visualizer </h1>

Program that helps to visualize how the 'Soring Algorithm' works.

<h3>How to clone & get live server working</h3>

## **clone the repository**

> Either create a fork or a branch ! on your prefrence! and clone the repository.

bash
```
 git clone yourBranch / fork
```

## **Enviroment Setup**

> Install the Node.js **Required version** - v22.14.0

>Open the cloned Repository and cd to sorting-visualizer

bash
```
cd sorting-visualizer
```
>Intall essentials node module!

bash
```
npm install
```
after install is complete try to run the developement server !

bash
```
npm start
```
